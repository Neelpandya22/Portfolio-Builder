# Portfolio-Builder
🚀 Portfolio Builder A dynamic Portfolio Builder using Node.js and React. Users can create, customize, and showcase portfolios with drag-and-drop sections, live preview, and responsive templates. Built with Express, MongoDB, and JWT authentication. Easily deployable on AWS
🔥 Features
✅ User Authentication (JWT-based)
✅ Customizable Portfolio Templates
✅ Drag-and-Drop Sections
✅ Live Preview & Editing
✅ Responsive & Modern UI
✅ Database Integration (MongoDB)
✅ Deployed on Cloud (AWS)

🛠️ Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB 
Authentication: JWT & OAuth
Deployment: AWS
